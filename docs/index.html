快速开始Quick Start 

# Clone the repository
git clone https://github.com/AutoZYX/Co4Pilot.git
cd Co4Pilot
./install.sh

安装过程采用追加模式, 不会影响现有开发环境。卸载: Append-only installation -- your existing dev environment stays untouched. To uninstall: ./install.sh --uninstall

关于About

Co4Pilot 由 张玉新 创建, 他是吉林大学汽车工程学院副教授, 研究方向为自动驾驶安全、SOTIF与场景驱动测试。 每个技能文件包含200-500行实质性内容, 框架和模板来源于 MIT、Stanford、Harvard、CMU、Oxford 等顶级大学的最佳实践。

Co4Pilot was created by Yuxin Zhang, Associate Professor at Jilin University, School of Automotive Engineering. Research: autonomous driving safety, SOTIF, scenario-based testing. Every skill file contains 200-500 lines of substantive content with frameworks and templates sourced from MIT, Stanford, Harvard, CMU, Oxford, and other top universities.

灵感来源于Inspired by automotive-claude-code-agents (Thejeswarareddy R, Bosch), 该项目展示了如何将领域专业知识系统化地集成到 AI 编程助手中。 by Thejeswarareddy R (Bosch), which demonstrated how domain-specific knowledge can be systematically integrated into AI coding assistants.
