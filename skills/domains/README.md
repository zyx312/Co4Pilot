# Domain Extensions

Co4Pilot's 4P framework (People/Project/Paper/Patent) provides universal academic research skills. **Domain extensions** add discipline-specific expertise on top of this foundation.

## How to Contribute a Domain Extension

If you're an expert in a specific research field, you can contribute a domain extension:

1. Create a folder under `skills/domains/your-domain/`
2. Add skill files (`.md`) following the Co4Pilot skill template
3. Each skill should contain: overview, frameworks, templates, prompt templates, references
4. Submit a PR with your domain added to this table
